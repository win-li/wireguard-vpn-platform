## 4. 业务规则

### 签到奖励
- 基础签到积分: 10分
- 连续7天奖励: 50分
- 连续14天奖励: 100分
- 连续30天奖励: 300分

### 任务奖励
| 任务 | 类型 | 积分 | 天数 |
|------|------|------|------|
| 每日签到 | daily | 10 | 0 |
| 首次登录 | one_time | 50 | 1 |
| 邀请好友 | achievement | 100 | 3 |
| 分享给好友 | daily | 5 | 0 |
| 完善个人资料 | one_time | 20 | 0 |

### 积分兑换规则
| 积分 | 奖励 |
|------|------|
| 100 | 1天会员 |
| 500 | 7天会员 |
| 1800 | 30天会员 |

### 佣金比例
- 一级佣金: 5%
- 二级佣金: 2%
